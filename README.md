# iPhone 15 Pro Website Clone

⚠️ Disclaimer:
This project is a frontend UI clone of the iPhone 15 Pro website,
created purely for learning and educational purposes.
All design credits belong to Apple.
No commercial use intended.

Inspired by JavaScript Mastery tutorial.

## Tech Stack
- React.js
- Vite
- Three.js
- GSAP (GreenSock Animation Platform)

## Features
- Apple-style smooth animations using GSAP
- 3D product visualization with Three.js
- Responsive and modern UI
- Component-based architecture with React
